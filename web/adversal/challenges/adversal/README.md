# Build

```
docker compose build
```

# Confirm Docker Image

```
docker images
```

Should see something like:

```
REPOSITORY        TAG       IMAGE ID       CREATED          SIZE
adversal          1         83cbce6baae1   17 minutes ago   405MB
```
